Math & Sciences Hub — Tutor Website Sync Setup

Files included:
1. modTutorWebsiteSync.bas
2. tutors_from_workbook.json
3. index_tutor_verification_fixed.html

How to connect Excel to the website:
1. Open your Excel system workbook.
2. Save a macro-enabled copy as .xlsm
3. Press Alt + F11
4. In VBA editor: File > Import File > choose modTutorWebsiteSync.bas
5. Run SetupTutorWebsiteSync once
6. In EXPORTS sheet:
   - I3 = website folder where index.html and tutors.json sit
   - I4 = tutors.json
   - I5 = export year
   - I6 = Yes to write missing verification codes back into TUTOR_REGISTER
   - I7 = Yes to include pending tutors in the JSON
7. Run ExportTutorsJson

Recommended flow:
- Update tutor data in TUTOR_REGISTER
- Run ExportTutorsJson
- Upload the refreshed tutors.json to your website folder / hosting

What the VBA exports:
- code
- displayName
- name
- role
- qualification
- institution
- subjects
- status

Important:
- The website must keep tutors.json in the same folder as index.html unless you change the fetch path in the site code.
- The fixed HTML file corrects a JavaScript error that would have blocked tutor verification loading.
